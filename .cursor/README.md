# Cursor Context for Anchor Learning

This directory provides context files to help AI assistants understand the learning structure and materials.

## Files

### learning-context.md
Quick reference for learning materials, roadmap, and practice projects. Use `@learning-context.md` in chat to provide context about the learning structure.

## Usage

When helping with learning:
1. Reference `@learning-context.md` for overview
2. Check `LEARNING_ROADMAP.md` for detailed learning path
3. Use `LEARNING_GUIDES/` for specific topic explanations
